# Decimal To Binary Converter 
A simple web app that converts decimal (base-10) numbers into binary (base-2) in real time.
Built with pure HTML, CSS, and JavaScript

## Overview

The Decimal to Binary Converter allows users to:
- Enter any decimal number into an input field
- Instantly convert it into its binary equivalent

## Tech Stack

- HTML: Structure of the app
- CSS: Styling and layout
- JavaScript: Conversion logic and interactivity

## Features

- Decimal number input
- One-click or automatic conversion to binary
- Input validation
- Responsive layout (works on desktop & mobile)

## How It Works

- The user enters a decimal number into the input field
- JavaScript reads the value
- The number is converted into binary using JavaScript logic
- The binary result is displayed instantly on the screen
